# EventMaster - Global Event Management Platform

EventMaster is a professional, international-standard frontend built with **React** for managing high-scale conferences, industry summits, and elite networking events.

## 🚀 Features

- **Professional UI/UX**: Designed with a corporate aesthetic, utilizing modern design patterns and animations.
- **Dynamic Events Directory**: Real-time search and category filtering for extraordinary global events.
- **Enterprise-Grade Auth**: Sophisticated login and registration flows with simulated loading states and validation.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **High Performance**: Built with React Suspense and Lazy Loading for optimal load times.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Routing**: React Router 7
- **Styling**: Vanilla CSS with a customized Design System
- **Icons**: Lucide React
- **Typography**: Plus Jakarta Sans

## 📦 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/event-management-frontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd event-management-frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
